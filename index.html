Byteball.be - maybe the best lightning node out there 

What is byteball.be?

Byteball.be is a Bitcoin lightning node hosted in the hearth of Switzerland providing strong uptime and reliance. Feel free to open a channel with bybeball.be

Why the name byteball.be?

Byteball.be wants to support the developpement and awarness of the Byteball cryptocurrency by establishing a node for the lightning network. 

Some useful features:

  1. Interactive source editor
  2. HTML Cleaning
  3. Word to HTML conversion
  4. Find and Replace
  5. Lorem-Ipsum generator
  6. Table to DIV conversion
